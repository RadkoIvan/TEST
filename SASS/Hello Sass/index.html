Імпорт файлів SCSS

Будь-який файл scss (стилі окремих сторінок, блоків, обнулюючий стиль і т.д.) файл можна імпортувати до основного файлу scss


Імпорт файлу scss: @import "nullstyle.scss";

Шаблони

Для елементів, блоків в документі зі стилями які повторюються можна використовувати шаблони.

Для створення шаблону на початку його назви слід поставити знак %. Таким чином ми створимо шаблон який не відображатиметься в файлі стилів css.

Також за допомогою функції доавання шаблонів можна створити не існуючий клас з потрібними для цього шаблону стилями, і додавати його до тих чи інших елементів документу.

Коментарі

В препроцесорі можна використовувати як стандартні коментарі css /* тут якийсь коментар*/, які переписуються з scss файлу в css файл, так ф коментарі препоцесору // тут коментар , який на відміну від попередьного варіанту не записуєтся в головний файл стилів.


Коментарі scss строчні, тобто діють до нового рядка в файлі:

// тут коментар
// тут інший коментар
